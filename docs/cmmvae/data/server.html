cmmvae.data.server

This module contains a LightningDataModule class that manages Datapipes and Dataloader creation for data from cellxgene-census.

Submodules:
  • SpeciesManager: Manages species Datapipe creation.
  • SpeciesDataModule: LightingDataModule for local npz, pkl dataset.
